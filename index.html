Simply Toast

Simple toast messages for jQuery (and friends)

View project onGitHub

simply-toast

jQuery (or equivalent) is required, all css styles were extracted from bootstrap, so, if you already use bootstrap, you don't need simply-toast.css

Samples (click on the code block)

$.simplyToast('This is a success message!', 'success');

$.simplyToast('This is a danger message!', 'danger');

$.simplyToast('This is a info message!', 'info');

$.simplyToast('This is also an info message!'); //'info' is the default type

$.simplyToast('This is a warning message!', 'warning');

Methods

// To create a toast message
$.simplyToast(message, type='info', options);

// To manually remove a toast message
$.simplyToast.remove(jqueryEl);

// To extend default options
$.extend(true, $.simplyToast.defaultOptions,
{
   'type': 'success'
});

Available options and default values

appendTo: "body",
customClass: false,
type: "info",
offset:
{
   from: "top",
   amount: 20
},
align: "right",
minWidth: 250,
maxWidth: 450,
delay: 4000,
allowDismiss: true,
spacing: 10
